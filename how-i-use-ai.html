How I Use AI04
Mode 01AI as my build partner

A repeatable loop, with a hard rule.

01 Me

Set intent & context

The what lives in markdown before the agent writes anything: architecture, users, voice, guardrails. Each prompt is pointed at the right files and scoped to one thing.

02 Agent

The agent writes the code

A rule into SQL, a tag taxonomy into columns, a panel wired to an endpoint, anywhere the shape is already decided and the typing is the only cost.

03 Me

Judge it, log what's wrong

The agent can build it two ways and can't tell me which is right. I decide. Whatever stays broken goes to TODO.md with the reason, so the loop never loses it.

How I Use AI05
Mode 01 · Context Engineering

Markdown is the working memory.

Every project runs on the same context files. They're how I keep an agent aligned across long sessions — the difference between prompting into the void and directing a build.

PLAN.md
Architecture and the hard technical decisions.
PRODUCT.md
Who it's for, scope, and product voice.
DESIGN.md
The design system and anti-slop rules.
CLAUDE.md
Guardrails that never change.
TODO.md
Known-broken, with the reason it broke.

Plus targeted agent skills that audit engineering and database-schema gaps, so the review is repeatable, not vibes.

How I Use AI06
Mode 01 · From Prototype to Product

Prototype fast, then engineer.

All three started as Lovable prototypes: fast to something real, impressive for so little direction. But a prototype isn't a product. The ones that mattered moved to Claude Code and real architecture.

Lovable: prove it

A vague prompt to a working end-to-end app in hours. The fastest way to find out whether an idea is even worth building, and to see what the interface actually needs.

Claude Code: build it

Markdown-driven and architecture-first: state out of pages, math into services, UI into small components. RoamsWild's 2,606→218-line refactor lived here.

How I Use AI08
RoamsWild · In-Product AI

AI as judgment over data.

Open data feeds, modern APIs, and deterministic geometry do the heavy lifting. AI is the judgment layer on top, and deliberately a secondary signal.

Gemini vision reads satellite imagery to assess whether a derived spot is actually worth the drive.
Plain-language reasoning across structured data, so every recommendation can explain itself.
A chat assistant answers "what's a good hike near me?" from the same filtered, legality-aware set.
The rule: every recommendation traces to a real fact first: a polygon, a sun vector, a road track — not the model.
How I Use AI09
ClipQuery · In-Product AI

Local-first, on your machine.

A media vault that runs AI entirely on-device, the opposite of cloud-everything. Nothing leaves your computer.

LLaVA via Ollama describes and tags scenes locally, swappable from llava:7b to 34b behind one constant.
CLIP embeddings power semantic search: find clips by what's in them, not the filename.
AI as an invisible utility: it generates structured data, it never dictates the aesthetic: no sparkle, no "magic" states.
Provenance on every field: [HW] hardware, [ML] model, [USER] verified. Edit an [ML] value and it locks to [USER].

How I Use AI11
What I Learned

Where AI breaks.

It collapses the cost of building — not the cost of being wrong. The hardest failures look correct from every angle.

Hallucinations

Invented table names and function signatures. The data lies too: OSM tags disagree, polygons drift. Surface the raw data early and most of it gets caught.

Context ceiling

Confident at 200 lines, useless at 2,600. A monolith the agent could no longer reason about, refactored back to 218. A discipline win, not an AI win.

Emergent bugs

The worst bugs are where every local piece is correct: a recursive RLS policy, a paved road tagged "4WD." Nothing the model sees is wrong.

How I Use AI12
The Through-Line

AI won't give you taste, trust, or restraint.

The model optimizes for a plausible screen. It won't make things consistent, honest, or restrained on its own — that part is still mine.

Consistency

Agents won't use a system or styled components by default. RoamsWild's "Pine + Paper" was the move the agent never got to make.

Provenance

Show where data comes from. ClipQuery declares [HW]/[ML]/[USER] and bans the "magic AI" sparkle states.

Restraint

Agentic tooling makes anything buildable fast, which turns scope into the hard problem. Constraint is a skill.
